# 🌡️ Temperature Monitoring System using Arduino UNO & DHT11

## 📌 Overview

This project demonstrates a simple temperature monitoring system using an Arduino UNO and DHT11 Temperature Sensor. The system continuously reads temperature data and turns ON an LED whenever the temperature exceeds a predefined threshold.

This project was completed as part of my Embedded Systems & IoT Internship.

---

## 🛠 Components Used

- Arduino UNO
- DHT11 Temperature Sensor
- LED
- 220Ω Resistor
- Breadboard
- Jumper Wires
- Wokwi Simulator

---

## ⚙ Features

- Reads real-time temperature
- Displays data on Serial Monitor
- LED alert when temperature > 30°C
- Simulated using Wokwi
- Beginner-friendly embedded systems project

---

## 🔌 Circuit Connections

| DHT11 | Arduino UNO |
|--------|-------------|
| VCC | 5V |
| DATA | D2 |
| GND | GND |

LED

- Anode → D13 (through 220Ω resistor)
- Cathode → GND
